= Authentication

Actions are integrated with authorization rules, providing a rich access control
layer between users and data.

Most authorization rules refer to the Subject of the request. To identify the
Subject, we need to authenticate each request.

Site supports two options for authenticating users:

* HTTP Authentication, as described in https://www.rfc-editor.org/rfc/rfc7235[RFC 7235], which we describe in <<ch-http-authentication>>.

* Cookie-based authentication, which we describe in <<ch-cookie-authentication>>.
